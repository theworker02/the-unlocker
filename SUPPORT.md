# Support

Use GitHub issues for reproducible bugs, documentation gaps, and feature proposals.

Please include:

- The command or workflow you ran.
- Your OS and .NET/Node/Ruby/Rust versions.
- Relevant logs or screenshots.
- Whether you are using the desktop app, frontend, Ruby API, Rust worker, or .NET registry.

Do not file requests for licensing bypasses, anti-cheat bypasses, authentication bypasses, integrity-check evasion, or runtime patching of protected third-party logic.
