# Example Mod Gallery

Use this gallery as the source for future sample mods:

- `menu-item-mod`: registers menu items.
- `asset-importer-mod`: registers asset importers.
- `settings-mod`: demonstrates typed settings.
- `event-mod`: publishes and subscribes to events.
- `theme-mod`: contributes theme resources.
- `command-mod`: registers command palette entries.
- `tool-panel-mod`: contributes a document/tool panel.

`SampleMod` is the first working example and should remain the smoke-test mod.
